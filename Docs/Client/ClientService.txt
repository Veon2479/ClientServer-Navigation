Класс службы клиента для запуска считывания данных с сенсоров и запуска математических
вычислений.
-------------------------------------
Ипользование:
	-MainActivity
-------------------------------------
Методы:
-----------<
	ClientService(Context sContext)
Конструктор.
-----------<
	int onStartCommand(Intent intent, int flags, int startId)
Инициализация службы при его запуске. Запускаются потоки сенсоров и математики.
