Класс наследует интерфейс Runnable. Определяет поток математических и физических вычислений.
Применяется фильтр Калмана.
-------------------------------------
Ипользование:
	-ClientService
-------------------------------------
Поля:
	public boolean fPhysics = true;
Флаг разрешения выполнения вычислений.
	public Quaternion tempQuat;
Промежуточный кватернион, для вычислений.
	public Quaternion rotQuat;
Кватернион поворота телефона.
	public Quaternion linAccQuat;
Кватернион полученный из вектора ускорения в координатах телефона. [ax, ay, az, 0].
	public double accX;
Ускорение относительно земли по X.
	public double accY;
Ускорение относительно земли по Y.
	private double time;
Текущее время.
    	private double deltaTime;
Время между итерациями фильтра.
<------------------------------------------>
	private Matrix H; 	- матрица наблюдений
	private Matrix Htr; 	- транспонированная матрицанаблюдений
	private Matrix P; 	- ковариационная экстраполяция 
	private Matrix P1;	- ковариационная экстраполяция предыдущей итерации
	private Matrix F;	- матрица перехода состояния
	private Matrix R;	- матрица погрешностей измерения
	private Matrix Q;	- матрица шумов.
	private Matrix K;	- Kalman Gain
	private Matrix I;	- Единичная матрица

	private Matrix predX;	- Предсказанное состояние системы
	private Matrix currX;	- Усреднённое состояние системы

	private Matrix z;	- Входная матрица измерений z = |ax|
								|ay|

Матрицы для работы с фильтром Калмана.
<------------------------------------------>
-------------------------------------
Методы:
-----------<
	UpdateGlobalAcc()
Преобразовывает вектор ускорения из локальных координат телефона в глобальные координаты,
где y - направлен на север, z - направлен перпендикулярно поверхности земли, x -  перпенди-
кулярен y и z.
-----------<	
	PhysicsProc()
Вычисления координат простейшей физикой. Необходим только для тестировки.
-----------<
	KalmanFilter()
Итерация фильтра калмана. На основании матриц сформированных на предудыщих итерациях и
измеренного ускорения вычисляет текущее состояние системы. Состояние системы определяется
вектором x = [x  vx  ax  y  vy  ay]^T.
-----------<
	run()
Реализация действий потока. Вычисляется время с прошедшей итерации и запускается фильтр
Калмана.
-----------<
	Disable()
Отключает поток.
-----------<
Дополнительно:
	В классе есть флаг fPhysics, при необходимости не отключая поток приостоновить
вычисления следует установить флаг в false.
